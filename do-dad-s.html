W3.CSS Containers

The container class is the most important of the W3.CSS classes. It provides equality like:

  • Common margins
  • Common paddings
  • Common vertical alignments
  • Common horizontal alignments
  • Common fonts
  • Common colors

The container class is typically used with HTML container elements, like:

<div>, <header>, <footer>, <article>, <section>, <blockquote>, <form>, and more.
